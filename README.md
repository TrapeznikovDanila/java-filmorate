# java-filmorate

Приложение, которое поможет определиться, какой фильм посмотреть вечером. Это социальная сеть, с возможностью 
пользователям добавлять фильмы, ставить лайки/дизлайки, получать рекомендации, находить друзей по интересам. Фильмы 
внутри приложения подразделяются по жанрам и рейтингу MPAA.

### Стек
- Java 18
- Spring Boot
- Lombok
- Maven
- Junit
- JDBC
- SQL
- H2

### Функциональность:
- Создание, редактирование и удаление профилей пользователей
- Получение списка всех пользователей
- Добавление/удаление друзей
- Получение списка друзей пользователя
- Получение списка общих друзей между двумя пользователями
- Получение списка рейтингов, получения рейтинга по id
- Получение списка жанров, получение жанра по id
- Создание и редактирование карточки фильма
- Возможность поставить/удалить лайк фильму
- Получение списка всех фильмов
- Получение фильма по id
- Получение списка наиболее популярных фильмов
